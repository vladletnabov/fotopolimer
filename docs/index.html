Виды фотополимеров

Фотополимеры отличаются между собой по ряду физических параметров: вязкость, твёрдость, относительное удлинение при разрыве, набухание в воде/спиртах/пластификаторах, остаточная адгезия, запах.

Фотополимер «Roehm» США. Твердость 50 по Шору.
Один из лучших фотополимеров по весьма привлекательной цене.
Имеет низкую остаточную адгезию (липкость), высокую разрешающую способность, малое время засветки, хорошо вымывается. Благодаря низкой вязкости данного полимера воздушные пузырьки легче выходят из него.

Фотополимер «Дихром Люкс» Неплохой плимер пр-ва России. Твёрдость 55.

Фотополимер «Colop VX 55» Австрия. Твёрдость 55. Канистра имеет удобную для наливания крышку.

Фотополимер «IDEAL i 50» США. Твёрдость 50.
